# rem-adapt

> A plugin to resize root element's fonsSize according to designWidth.

# Usage

```bash
npm install rem-adapt
```

```javascript
import remAdapt from 'rem-adapt'
remAdapt(1080, 100) // 设计稿宽度标准为1080px，px2rem比例为100
```

```less
.baner {
  width: 1rem; // 100px
  height: 1.8rem; // 180px
}
```
